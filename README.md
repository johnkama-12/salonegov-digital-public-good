# SaloneGov Digital Public Good System

SaloneGov is a proposed Sierra Leone Government Digital Services Portal. It allows citizens, businesses, foreign nationals and government staff to access public services from one secure online platform.

## Project Type
PROG 102 - Principles of Software Engineering Final Project: Software Design for a Digital Public Good.

## Main Features
- User registration and login
- Browse government services by category
- Submit online applications
- Upload supporting documents
- Make service payments where required
- Track application status
- Receive notifications
- Government staff review and approval dashboard

## SDG Alignment
- SDG 16: Peace, Justice and Strong Institutions
- SDG 9: Industry, Innovation and Infrastructure
- SDG 10: Reduced Inequalities
